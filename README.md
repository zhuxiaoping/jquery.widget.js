jquery.widget.js
================
  
简介
-----------------------------------
  手机端简单widget
  
### 参数
    speed /*窗口滑出速度，单位ms，默认300ms*/
    width /*窗口滑出距离，默认body的宽度*/
    noCloseClass /*class名称，widget包含此class，则点击不关闭，默认widget-noclose*/
    widgetBodyClass /*内容主体class名称，默认widget-body*/
    onClose /*窗口关闭后调用*/
    showRightToLeft /*窗口滑出方向，默认true，表示默认从右往左滑出*/
